Project Showcase

Influencer Engagement & Sponsorship Coordination Platform - V2 (Advanced Version)

Completed: December 2024

Abstract

The project focuses on the development of an advanced web application platform designed to bridge the gap between sponsors and influencers. This advanced version builds upon the previous platform (V1) by adding features such as asynchronous task processing, enhanced performance using Redis caching, and improved security with token-based authentication. The system allows sponsors to create and manage campaigns, while influencers can accept ad requests and manage their profiles. The platform aims to streamline the advertising process, optimize influencer engagement, and provide better control over campaign outcomes.

Project Description

The "Influencer Engagement & Sponsorship Coordination Platform - V2" is an advanced version of its predecessor (V1), with several enhancements aimed at improving system performance and user experience. This version incorporates asynchronous task handling with Celery and Redis, allows real-time updates using Vue.js, and enhances security with JWT authentication. The platform facilitates campaign creation, management, and tracking, as well as influencer engagement, and aims to provide seamless interactions for both sponsors and influencers.

Project Objectives

  • Develop a platform for sponsors and influencers to engage in advertisement contracts (advanced version of V1).
  • Improve system performance by integrating Celery and Redis for task execution and caching.
  • Enhance security using JWT authentication and implement role-based access control.
  • Provide better user experience with real-time updates and performance optimizations.

Methodology

The project was developed following an agile methodology with the following steps:

  1. System architecture design with Flask backend and Vue.js frontend, building on the V1 framework.
  2. Integration of Celery and Redis for batch task execution such as sending reminders and generating reports, optimizing system performance.
  3. Frontend development with Vue.js for dynamic interaction, form validation, and real-time updates, improving the user interface.
  4. Database setup with SQLite, optimizing queries and adding caching mechanisms to handle increased load efficiently.

Technologies Used

Flask Vue.js SQLite Celery Redis JWT Bootstrap Python HTML CSS

Results

The platform is fully functional and operates more efficiently than the previous version. It allows seamless interaction between sponsors and influencers and supports campaign creation, ad requests, and user profiles. The advanced version incorporates significant improvements such as a 40% reduction in response times due to Redis caching, enhanced security with JWT authentication, and better scalability with the use of Celery for task execution.
