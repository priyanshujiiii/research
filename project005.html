August 2024

Abstract

This project aims to leverage a comprehensive dataset of criminal activities within a city to predict crime categories using machine learning techniques. The dataset includes various features such as the date, time, location, and victim demographics of each incident. By applying machine learning models, this project seeks to uncover patterns in crime occurrences and develop predictive models that can assist law enforcement in enhancing their strategies and improving public safety. The overall objective is to provide actionable insights for anticipating crime trends, improving crime prevention efforts, and optimizing resource allocation. By utilizing advanced data analysis methods, this project demonstrates how data-driven approaches can be applied to address complex social issues like crime prediction and safety management. One of the top-performing models in this project is the XGBoost Classifier, which reported an accuracy of 0.923 on the training dataset. However, when tested on the actual dataset, the model's accuracy slightly dropped to 0.88060. Despite this, XGBoost still demonstrates robust performance, indicating that it is highly capable of making accurate predictions for crime categories based on the features provided in the dataset.

Project Objectives

The objective of this project is to develop and implement optimized machine learning models that achieve high accuracy on the test data. The goal is to ensure robust performance and reliable predictions by applying advanced modeling techniques and thorough evaluation methods.

Methodology

The project utilized a preprocessed dataset of customer reviews and implemented the following steps:

  • Data Loading: Import the dataset and load it into the environment for analysis.
  • Data Exploration:
    • Understand the columns and their descriptions.
    • Analyze the data types in the crime dataset.
    • Check for outliers and missing values.
    • Visualize relationships using heat maps and other plots.
  • Feature Engineering: Drop irrelevant columns and engineer new features as needed for better predictions.
  • Data Preprocessing:
    • Preprocess the training and test data separately.
    • Split the data into training and validation sets.
  • Training the Model:
    • Build a baseline model using KNN.
    • Train various models and check for the best accuracy.
    • Decode predictions and make adjustments for final outputs.
  • Submission File: Generate the predictions and create a submission file for evaluation.
  • Results and Learning Outcomes: Evaluate the results, document findings, and derive insights for further improvement.

Results

The models generally exhibit strong reported accuracy, but there is a consistent decline in performance on the actual test dataset, with XGBoost (0.88060) and LightGBM (0.87220) leading the pack, while Gradient Boosting Classifier (0.83960) shows signs of overfitting despite the highest reported accuracy.
