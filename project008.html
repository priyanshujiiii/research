Abstract

The "Influencer Engagement and Sponsorship Coordination Platform" bridges the gap between sponsors and influencers, offering an efficient system for campaign management and ad request handling. This project showcases modern web development practices, integrating secure user authentication, robust data management, and scalable architecture.

Project Description

A dynamic web application catering to three roles: admin, sponsors, and influencers. The platform enables sponsors to create and manage campaigns, influencers to browse and handle ad requests, and admins to monitor platform activities. Key features include campaign categorization, role-based authentication, and robust search functionalities.

Project Objectives

The primary objectives of the project are as follows:

  • Facilitate sponsor-influencer collaboration.
  • Streamline campaign management and ad request handling.
  • Provide robust search and discovery functionalities.
  • Ensure secure and scalable platform operations.
  • Enable comprehensive admin oversight of platform activities.

Methodology

  1. Requirement analysis to define user roles and interactions.
  2. Database design using SQLite for campaigns, ad requests, and profiles.
  3. Backend development with Flask for role-based operations and data handling.
  4. Frontend development using Jinja2 templates and Bootstrap for UI design.
  5. Feature implementation for campaign and request management.
  6. Testing and validation through unit and integration tests.
  7. Deployment for local testing and demonstration purposes.

Technologies Used

Python Flask SQLite Jinja2 Bootstrap HTML CSS

Results

Successfully developed a functional platform facilitating seamless collaboration between sponsors and influencers. Achieved high responsiveness, scalability, and efficient data handling for user roles and campaigns.
