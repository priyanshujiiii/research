Methodology

The project utilized a preprocessed dataset of customer reviews and implemented the following steps:

  1. Data cleaning and preprocessing using Python libraries such as NLTK and SpaCy.
  2. Feature extraction with TF-IDF and word embeddings (GloVe).
  3. Model training using Logistic Regression, Random Forest, and fine-tuned BERT.
  4. Deployment of the best-performing model via Flask API.

Technologies Used

Python TensorFlow BERT Flask NLTK SpaCy SQLite HTML CSS

Results

Achieved an overall accuracy of 92% on the test dataset. The application can process over 500 reviews per second in real-time and provides insightful visualizations of sentiment trends.
