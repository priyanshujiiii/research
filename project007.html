Abstract

The project focuses on the development of an advanced web application platform designed to bridge the gap between sponsors and influencers. The system allows sponsors to create and manage campaigns, while influencers can accept ad requests and manage their profiles. The platform incorporates features such as role-based authentication, campaign management, ad request tracking, and batch processing for automated tasks. It aims to streamline the advertising process, optimize influencer engagement, and provide better control over campaign outcomes.

Project Description

The "Influencer Engagement & Sponsorship Coordination Platform - V2" is an advanced web application that connects sponsors with influencers to facilitate digital advertising. The platform allows sponsors to create, manage, and track campaigns while providing influencers the ability to respond to ad requests, negotiate terms, and update their profiles. Key features include role-based access control (RBAC), real-time notifications, and batch jobs for reminders and reports. Built using Flask for the backend, Vue.js for the frontend, and SQLite for data storage, the system is highly scalable and efficient.

Project Objectives

  • Develop a platform for sponsors and influencers to engage in advertisement contracts.
  • Implement role-based access control (RBAC) to manage user permissions effectively.
  • Provide asynchronous task execution with Celery and Redis for performance optimization.
  • Incorporate automated daily reminders and monthly activity reports for users.
  • Enhance security and privacy through token-based authentication and CORS handling.

Methodology

The project was developed following an agile methodology with the following steps:

  1. System architecture design with Flask backend and Vue.js frontend.
  2. Integration of role-based authentication using JWT for secure login and session management.
  3. Implementation of Celery and Redis for batch task execution such as sending reminders and generating reports.
  4. Frontend development using Vue.js for dynamic interaction, form validation, and real-time updates.
  5. Database setup with SQLite, storing data about users, campaigns, and ad requests.
  6. Performance enhancement through caching techniques using Redis.

Technologies Used

Flask Vue.js SQLite Celery Redis JWT Bootstrap Python HTML CSS

Results

The platform is fully functional, allowing seamless interaction between sponsors and influencers. It supports the creation of multiple campaigns, ad requests, and user profiles. The system successfully automates batch tasks such as daily reminders and monthly reports. Performance tests showed a 40% reduction in response times due to the implementation of Redis caching. Security measures, including token-based authentication, ensure safe user interactions.
